Information

As part of the development of the master's dissertation, this program was developed with the objective of identifying the movement of a given object, in real time, in the environment of a game, using Machine Learning and Computer Vision techniques, more specifically methods of Object Detection and Object Tracking

The practical environment was developed using the OpenCV python library, which has a diverse range of computer vision algorithms available and also allows the parallel use of CPU and GPU for the optimization of these same algorithm.

To start the program it is necessary to choose a window, an image and a threshold value and then click on start.The program will search for the provided image in screenshots constant to the window, highlighting only the objects found with equality equal to or greater than the given threshold.

Configurations

Window Capture Choose the window you want the program to capture. It needs to have the exact name of the window for the program to be able to recognize it. It will take constant screenshots of the window that will be used in template matching.
In case of error, maximize the window in question and re-select the window (don't minimize the selected window).

Template Matching Choose the image you want the program to detect on the window selected above. It cannot be an image without a background. Take, for example, a screenshot of the object you want and save the image.
If the image has too much background, the program may try to detect the background instead of the object. Try to make the image contain only the object, or a large part of it.

Threshold Corresponds to the minimum degree of equality necessary to determine whether the provided image and the found object represent the same thing. If they are the same, the object is highlighted.
If no objects are detected, try to decrease the threshold. If too many objects are detected, increase its value.

When running press 'q' to exit
